Introducción

Bienvenido al Objeto Virtual de Información (OVI) de Diseño de Sitios Web, diseñado para que pueda adquirir los conocimientos básicos para el uso de GIT, Github y Github Pages para la gestión de proyectos colaborativos y el uso de HTML5 y CCS3 para la creación de contenido en formato web (Página y Sitios Web)

Para acceder a los contenidos de manera secuencial puede hacer clic en el botón siguiente, si desea acceder a un recurso en específico puede buscarlo en el menú superior.

Objetivos del OVI

Construir un OVI para introducir, profundizar sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, al igual que HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web)

Objetivos del Curso

  • Enseñar al lector del OVI los conceptos básicos para el uso de GIT y GITHUB para crear y gestionar proyectos colaborativos de diseño web
  • Enseñar al lector del OVI los conceptos básicos para el uso del lenguaje de etiquetas HTML5 para la creación de contenido WEB
  • Enseñar al lector del OVI los conceptos básicos para el uso del lenguaje de estilos CCS5 para la creación estilos para el contenido WEB
